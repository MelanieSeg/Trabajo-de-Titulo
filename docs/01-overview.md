# 1. Visión General

## Objetivo

EcoEnergy Platform es un sistema web para gestionar consumo de electricidad y agua en organizaciones con múltiples instalaciones, permitiendo:
- Centralizar métricas históricas de consumo y costos.
- Visualizar KPIs y tendencias con dashboards interactivos.
- Detectar variaciones anómalas y eventos críticos.
- Generar predicciones con modelos de aprendizaje automático.
- Definir metas de eficiencia y monitorear cumplimiento.

## Alcance funcional

- Ingesta de datos mensuales por CSV (ETL).
- Persistencia relacional en PostgreSQL.
- Analítica histórica y agregaciones por período.
- Predicciones ML de corto plazo (horizonte configurable).
- Alertas inteligentes por umbral y volatilidad.
- Exportación y reportabilidad para decisiones operativas.

## Usuarios objetivo

- Equipo de sostenibilidad.
- Operaciones industriales/planta.
- Finanzas (control de costos energéticos).
- Liderazgo con foco en eficiencia y reducción de huella.

## Valor del sistema

- Menor tiempo de análisis manual.
- Detección temprana de desviaciones de consumo.
- Priorización de acciones correctivas.
- Base trazable para auditorías energéticas.
